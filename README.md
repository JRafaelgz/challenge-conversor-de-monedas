  # Conversor de Moneda | Challenge ONE - Oracle Next Education - Java - Back end
---
Crear una aplicación en la terminal de Java que realice el cambio de moneda. Se debe comunicar a la API [ExchangeRate-API][link_exchangeAPI], que retorna el cambio entre dos monedas.
---

Pasos para completar el reto:  

  - Configuración del Ambiente Java;
  - Creación del Proyecto;
  - Consumo de la API;
  - Análisis de la Respuesta JSON;
  - Filtro de Monedas;
  - Exibición de Resultados a los usuarios;
